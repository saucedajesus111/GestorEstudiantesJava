# Gestor de Empleados en Java

Este es un programa sencillo en Java para registrar empleados dentro de un departamento.

## Conceptos usados

- Clases
- Objetos
- Atributos
- Constructores
- Metodos
- ArrayList
- Comentarios en el codigo

## Estructura

```text
GestorEmpleadosJava/
└── src/
    ├── Main.java
    ├── Empleado.java
    └── Departamento.java
```

## Como ejecutar

Desde la carpeta del proyecto:

```bash
javac src/*.java
java -cp src Main
```

## Que hace el programa

El programa crea empleados, los agrega a un departamento, muestra la informacion, aumenta el sueldo de un empleado, cambia el puesto de otro y busca un empleado por su nombre.
